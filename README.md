## ðŸ›  Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account
- Image storage (ImageKit/Cloudinary)

### Installation

1. **Clone and Install**
   ```bash
   npm install
   ```

2. **Configure Environment**
   Set up your `.env` files in `backend/`, `frontend/`, and `admin/` using the provided templates.

3. **Run Locally**
   - Backend: `cd backend && npm run dev`
   - Frontend: `cd frontend && npm run dev`
   - Admin: `cd admin && npm run dev`
